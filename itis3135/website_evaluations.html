Website Evaluations

Yale School of Art

This is Yale University's art school website with links and information pertaining to the school. This is an example of a bad website.

  • Contrast: Yale has the background of text highlighted which is easy to read, but lacks clarity as the colors change throughout.
  • Repetition: Yale uses the same gradient background for each section which is consistent throughout.
  • Alignment: Yale's alignment is all left which leaves too much open space on the right.
  • Proximity: Yale has everything in sections, but they aren't labeled well and contain lots of hard to read text.

Unseen Studio

Unseen Studio is a website design agency. This is an example of a good website.

  • Contrast: Unseen makes the wording contrast with the background so it is never hard to read.
  • Repetition: Unseen doesn't use the same thing over and over, rather the same layout for each section letting the website flow.
  • Alignment: Unseen centers their work and has everything the same size.
  • Proximity: Unseen groups the different designs by color meaning all the blues, greens, etc are grouped together for ease of use.
